# GoDefer

A `no_std` implementation of [Go's `defer` statement](https://tour.golang.org/flowcontrol/12), which executes a block of code when the surrounding scope ends.

## Usage

Add the dependency in your `Cargo.toml`:

```toml
[dependencies]
godefer = "1.0.0"
```
